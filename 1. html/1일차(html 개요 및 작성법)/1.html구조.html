html문서의 특징

1. 태그 내부에 또 다른 태그 사용이 가능하다
2. 태그엔 대소문자 구분이 없다
3. 파일 저장시 확장자는 -> xxx.html or xxx.htm
4. 태그 내부에 속성이 존재한다 -> 태그의 자세한 설정이 필요할 때

5. html문서 -> 문자, 이미지, 영상, 음악 등 포함 가능
6. 줄바꿈 -> p(단락 구분), br(단독태그)
7. 보통 문자보다 한 단계 작은문자를 출력하고 싶어요
8. 비주얼 스튜디오 코드의 기능 : 태그명을 쓰고 엔터치면 자동으로 태그명입력
9. 삭제
10. 추가
11. 크아악윗첨자
12. 아래첨자
13. 하이라이트효과
14. 일반적인 글꼴, 크기, 색깔(CSS로 해결)

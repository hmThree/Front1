visual studio code에서 자주 사용되는 단축키

  1. html문서 템플릿 작성법 -> html파일 작성 후 ! 엔터 or html:5 엔터
  2. 태그 작성법 -> 태그 이름 엔터
  3. 태그 여러개 생성법 -> 태그명*n
  4. 태그 중첩 생성 태그명>태그명>태그명...
  5. 인접한 태그 생성하는 방법 -> 태그명+태그명2...
  6. 아이디설정 : 태그명#아이디명
  7. 클래스 선택자 설정 -> 태그명.클래스명(div.test)
  8. 태그{입력하고 싶은 텍스트}입력 후 엔터or탭키
  9. 숫자 삽입(1씩 자동증가) ex)div#name&*n(아이디, 클래스, 텍스트 사용)
  10. ^입력시 상위 계층으로 이동(중첩된 상황에서 상위계층으로 이동)
  11. ^^p 한단계 아래로 이동
  12. (태그..)그룹으로 묶는 경우 : div>(ul>li)+(ol>li)
  13. 자동완성기능 div#myid.classA.classB입력 후 엔터친 경우 동시에 div태그에 id, class선택자를 만듦
